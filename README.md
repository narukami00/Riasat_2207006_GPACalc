# GPA Calculator (JavaFX)

A lightweight JavaFX application to calculate semester GPA. Add courses with credits and grades, edit or remove them, and view real-time GPA.

## Features
- Add, edit, delete courses
- Automatic GPA calculation
- Export results as PDF
- Input validation

## Requirements
- Java 21+
- JavaFX SDK

## Run
Clone the repo and run with Maven:

```bash
git clone https://github.com/narukami00/Riasat_2207006_GPACalc.git
cd Riasat_2207006_GPACalc
./mvnw javafx:run
